# 外汇策略工坊

中文版 PC 端 EA 策略分析与构建工具原型。

## 启动

在当前文件夹运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\start-site.ps1
```

然后打开：

```text
http://127.0.0.1:5173/
```

## 已实现

- 可视化构建器：新增模块、拖动节点、编辑参数、复制/删除节点、自动布局。
- 策略分析：基于画布、风控、重大事件规则实时评分。
- 风控中心：新增规则、启用/停用规则、本地保存。
- 重大事件：新增事件、导入 CSV/JSON、导出事件规则。
- 回测报告：导入 HTML/CSV/TXT，提取常见指标和交易样本。
- 代码/导出：生成 MQL5 示例代码，导出项目 JSON。
- 本地保存：项目状态保存到浏览器 localStorage。
